Lógica de Programação com Python

Introdução

Aprenda a resolver problemas de forma lógica usando a linguagem Python. Aqui você entenderá como criar soluções eficazes com estruturas simples e claras.

O que é Lógica de Programação?

A lógica de programação é a sequência de passos que você cria para resolver problemas utilizando um computador. Em Python, você pode aplicar essa lógica de maneira eficiente e intuitiva.

Estruturas de Controle em Python

Com Python, você pode usar várias estruturas para controlar o fluxo de execução do seu programa. As principais estruturas são:

  • Condição (if-else): Utilizada para tomar decisões no código.
  • Laços de Repetição (for, while): Usados para repetir ações várias vezes.
  • Funções: Permitem organizar o código em blocos reutilizáveis.

Exemplos Práticos em Python

Exemplo 1: Condicional (if-else)

idade = 18
if idade >= 18:
    print("Você é maior de idade!")
else:
    print("Você é menor de idade!")

Este código verifica se a pessoa tem 18 anos ou mais. Se for, imprime "Você é maior de idade!", caso contrário, imprime "Você é menor de idade!".

Exemplo 2: Laço de Repetição (for)

for i in range(1, 6):
    print(f"Contagem: {i}")

Este código imprime os números de 1 a 5 utilizando um laço "for".

Exemplo 3: Função

def saudacao(nome):
    print(f"Olá, {nome}!")

saudacao("Maria")

Este código define uma função chamada saudacao que recebe um nome como parâmetro e imprime uma saudação personalizada.
