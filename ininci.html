Desenvolvimento de Sistemas para Internet (DSI)

O curso de Sistemas para Internet é uma formação que capacita os estudantes para atuar com criação e manutenção de soluções tecnológicas voltadas para a web. Ao longo dessa formação, os alunos mergulham em um ambiente de aprendizado que abrange tanto os fundamentos teóricos quanto as habilidades práticas necessárias para desenvolver sistemas eficientes e inovadores.

- O que aprendemos:

HTML é uma linguagem de marcação, onde abaixo está a estrutura básica:

  • HEAD: É onde fica toda a parte inteligente da página. No HEAD ficam os metadados. Metadados são informações sobre a página e o conteúdo ali publicado.
  • meta charset: É uma metatag responsável por chavear qual tabela de caracteres a página está utilizando
  • title: É o texto que fica na barra de título do navegador
  • body: É onde todo o conteúdo visível no navegador deverá ser colocado.

Esse foi uma brévia do que aprendemos esse ano na matéria DSI
